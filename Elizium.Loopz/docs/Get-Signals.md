# Get-Signals

## SYNOPSIS

Returns a copy of the Signals hashtable.

## SYNTAX

```powershell
Get-Signals [[-SourceSignals] <Hashtable>] [[-Custom] <Hashtable>] [<CommonParameters>]
```

## DESCRIPTION

The signals returned include the user defined signal overrides.
